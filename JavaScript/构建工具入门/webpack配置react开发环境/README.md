## webpack配置react开发环境  
### 说明 
```  
最近在学习React，就使用webpack配置了react开发环境。  
使用：  
1、输入"webpack-dev-server"进行热更新。  
2、输入"webpack"打包文件。  
其他：  
js文件、css文件均已压缩。
```
### 生成目录  
```
.
|-- js                            // js文件目录
|-- css                           // css文件目录
|-- img                           // 图片
|-- index.html                    // 程序入口html文件
.
```
